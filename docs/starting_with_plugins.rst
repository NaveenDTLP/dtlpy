Starting With Plugins
=====================

Write something

Create a plugin
---------------

Check out a working project

.. code::

   $ dlp projects checkout -p "Animals"

Check out a working dataset

.. code::

   $ dlp dataset checkout -p "Walrus"


Create a Plugin from current directory

.. code::

   $ dlp plugins init

Test the code locally
---------------------

.. code::

   $ dlp plugins test

Deploy to cloud
---------------

.. code::

   $ dlp plugins deploy
